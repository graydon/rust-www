Rust installieren

Um Rust zu installieren, führe den folgenden Befehl in deinem Terminal aus und folge dann den Anweisungen auf dem Bildschirm.

curl https://sh.rustup.rs -sSf | sh

Um Rust zu installieren, lade den folgenden Installer herunter: rustup‑init.exe und folge dann den Anweisungen auf dem Bildschirm.

Hinweise zur Rust-Installation

Toolchain Management mit rustup

Rust wird vom rustup Tool installiert und verwaltet. Rust hat einen sechswöchigen schnellen Release-Prozess und unterstützt eine große Anzahl an Plattformen , somit sind viele Versionen des Rust-Compilers jederzeit verfügbar. rustup verwaltet diese Versionen in einem einheitlichen System auf jeder Plattform, die von Rust unterstützt wird und verwaltet zusätzlich die Installation von Beta- und Nightly Release Channels, sowie zusätzliche Cross-Compilation Versionen.

Für weitere Informationen lies dir bitte die rustup Dokumentation durch.

Die PATH-Umgebungsvariable konfigurieren

In der Rust-Entwicklungsumgebung sind alle Tools im Ordner ~/.cargo/bin %USERPROFILE%\.cargo\bin zu finden - hier findest du die Rust-Toolchain, inklusive rustc, cargo, und rustup.

Dementsprechend ist es für Rust-Entwickler üblich, diesen Ordner zur PATH-Umgebungsvariable hinzuzufügen. Während der Installation wird rustup, versuchen, die PATH-Variable automatisch zu konfigurieren, allerdings kann es, aufgrund Unterschieden zwischen verschiedenen Plattformen und Kommandozeilen, sowie Fehlern in rustup dazu kommen, dass Änderungen an der PATH-Variable nicht in Effekt treten, bis die Konsole / das Terminal neu gestartet wurde oder der Benutzer abgemeldet wurde. In seltenen Fällen kann es sein, dass die Änderung fehlschlägt.

Hinweise für Windows

Auf Windows benötigt Rust zusätzlich die C++ Build-Tools für Visual Studio 2013 oder neuer. Der einfachste Weg, diese zu installieren ist es, die Microsoft Visual C++ Build Tools 2017 zu installieren, welche nur die Visual C++ Build-Tools installiert. Alternative kannst du ebenfalls Visual Studio 2017, Visual Studio 2015, or Visual Studio 2013 installieren und während der Installation "C++ Tools" auswählen.

Für mehr Informationen über die Konfiguration von Rust auf Windows, bitte lies dir die Windows-spezifische rustup Dokumentation durch .

Andere Installationsmethoden

Die Installation durch rustup, wie oben beschrieben, ist der bevorzugte Weg für die meisten Entwickler, Rust zu installieren. Allerdings kann Rust auch über andere Wege installiert werden.
