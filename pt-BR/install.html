Instale Rust

Para instalar Rust, rode o seguinte no seu terminal e então siga as instruções na tela.

curl https://sh.rustup.rs -sSf | sh

Para instalar Rust, baixe e rode rustup‑init.exe e então siga as instruções na tela.

If you're a Windows Subsystem for Linux user run the following in your terminal, then follow the onscreen instructions to install Rust.

Notas sobre a instalação de Rust

Gerenciamento de toolchain com rustup

Rust é instalada e gerenciada pela ferramenta rustup . Rust tem um processo de lançamento rápido de 6 semanas e suporta um grande número de plataformas , então existem muitas versões de Rust disponíveis ao mesmo tempo. rustup gerencia estas versões de maneira consistente em todas as plataformas que Rust suporta, permitindo instalar Rust dos canais de lançamento beta e nightly e suporta mais alvos de compilação cruzada.

para mais informações veja a documentação de rustup .

Configurando a variável de ambiente PATH

No ambiente de desenvolvimento Rust, todas as ferramentas são instalados no diretório ~/.cargo/bin %USERPROFILE%\.cargo\bin , e aqui você encontrará a toolchain Rust, incluindo rustc, cargo, e rustup.

Portanto, é comum que desenvolvedores Rust incluam este diretório na sua variável de ambiente PATH . Durante a instalação rustup tentará configurar a variável PATH mas, devido às diferenças entre plataformas, shells de comando, e bugs no rustup, as modificações a PATH podem não ter efeito até que o terminal seja reiniciado, ou o usuário faça logout, ou podem simplesmente não ter sucesso.

Se, depois da instalação, executar rustc --version no terminal falhar, esta é a razão mais provável.

Considerações no Windows

No Windows, Rust requer ainda as ferramentas de build de C++ para Visual Studio 2013 ou posterior. A maneira mais fácil de obter as ferramentas de build é instalando Microsoft Visual C++ Build Tools 2017 que provê apenas as ferramentas de build Visual C++. Alternativamente, você pode instalar Visual Studio 2017 ou Visual Studio 2015 ou Visual Studio 2013 e durante a instalação selecionar "C++ tools".

Para mais informações sobre configurar Rust no Windows veja a documentação específica para windows do rustup .

Outros métodos de instalação

A instalação descrita acima, via rustup, é a maneira preferida de instalar Rust para a maioria dos desenvolvedores mas Rust pode ser instalada por outros métodos também.
