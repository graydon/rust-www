Установка Rust

Чтобы установить Rust, запустите эту команду в своём терминале и следуйте инструкциям на экране.

curl https://sh.rustup.rs -sSf | sh

Чтобы установить Rust, загрузите и запустите rustup‑init.exe и следуйте инструкциям на экране.

Дополнительная информация по установке Rust

Управление инстурментами с помощью rustup

Rust устанавливается и управляется инструментом rustup. Rust имеет 6-недельный процесс быстрого выпуска. и поддерживает обширное количество платформ , поэтому в любой момент доступны несколько билдов Rust. rustup управляет этими билдами в последовательном порядке на каждой платформе которую поддерживает Rust, обеспечивая установку Rust как с бета и nightly сборок, так и поддержу для дополнительных целей кросс-компиляции.

Для дополнительной информации см. rustup документацию.

Настройка переменной среды PATH

В среде разработки Rust, все инструменты устанавливаются в папку ~/.cargo/bin %USERPROFILE%\.cargo\bin , и это то место, откуда вы можете запустить инструментарий Rust, включая rustc, cargo, и rustup.

Соответственно, это нормально для Rust разработчиков включать этот путь в переменную среды PATH . Во время установки, rustup, попытается настроить PATH, но из-за отличий между платформами, командными оболочками и багами в rustup, изменения в PATH могут не вступить в силу пока консоль не будет перезапущена, или пользователь не выйдет из сессии, или могут не вступить в силу вообще.

Если попытка запустить rustc --version в консоли заканчивается неудачей, то скорее всего это наиболее вероятная причина.

Windows

На Windows, Rust дополнительно требует набор сборки C++ для Visual Studio 2013 или более поздней версии. Самый простой спосбов получения инструментов сборки - это загрузить Microsoft Visual C++ Build Tools 2017 который предоставляет только инструменты сборки Visual C++. Альтернативно, вы можете установить Visual Studio 2017, Visual Studio 2015, или Visual Studio 2013 и во время установки выбрать "C++ tools".

Для дальнейшей информации о настроке Rust на Windows, см. rustup документацию.

Другие способы установки

Установка, описаная выше, через rustup, это предпочтительный способ установки Rust для большинства разработчиков, но Rust может быть установлен и другими способами.
